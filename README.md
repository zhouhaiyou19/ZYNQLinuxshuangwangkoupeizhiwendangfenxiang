## 文档内容概述

该文档详细描述了如何在ZYNQ Linux平台上配置双网口，并特别说明了以下关键点：

- **双网口配置**：如何在ZYNQ平台上同时启用两个以太网接口。
- **MDIO共用**：两个网口如何共用MDIO接口进行管理。
- **RESET-GPIO不共用**：两个网口的RESET-GPIO信号不共用，分别独立控制。
